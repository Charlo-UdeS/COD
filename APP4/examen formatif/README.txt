Pour compiler les fichiers de l'examen formatif, il faut faire comme pour compiler les fichiers de la problématique. Après avoir créé le fichier "formatif-fonctions.c", tel que demandé dans l'énoncé du formatif, il faut changer la commande "construire" de Geany pour ce qui suit:

gcc -Wall -o formatif-main formatif-main.c formatif-fonctions.c

L'exécutable se nomme alors "formatif-main.exe"
